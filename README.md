# DeployButton

Это приложение представляет собой систему для управления звуковыми уведомлениями и эффектами с возможностью удаленного запуска через веб-интерфейс и физическую кнопку Arduino. Проект включает в себя веб-API, Angular-клиент и Arduino-прошивку для управления звуками.

## Описание

Приложение позволяет:
- Воспроизводить различные звуковые эффекты через веб-интерфейс
- Управлять звуками с помощью физической кнопки, подключенной к Arduino
- Автоматически устанавливать службу Windows для фоновой работы
- Обновлять аудио-файлы по требованию

## Структура проекта

- `DeployButton.Api` — ASP.NET Core Web API с SignalR-хабами для обработки запросов и управления воспроизведением звуков
- `ClientApp` — Angular-приложение с веб-интерфейсом для управления звуками
- `arduino_button/arduino_button.ino` — прошивка для Arduino для обработки физической кнопки
- `Mp3Formatter` — служебное приложение для обработки аудиофайлов
- `scripts/` — скрипты для установки и обновления службы Windows
- `raw_sounds/` — каталог с аудиофайлами

## Сборка и запуск

Для сборки проекта доступны PowerShell-скрипты:
- `build-all.ps1` — полная сборка всего проекта
- `build-api.ps1` — сборка только API
- `build-client.ps1` — сборка только клиентского приложения
- `build-sounds.ps1` — обработка аудиофайлов
